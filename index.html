Wrong Answer 答案错误释义 你的程序没问题,但是运行结果错了 解决办法 检查程序逻辑 检查数组越界 检查变量数据访问 检查括号匹配(自动匹配括号的可忽略) Time Limit Exceed 超时释义 你的程序陷入了死循环,或者循环次数过多,反正就是超时了 解决办法 检查所有循环 检查有无符合死循环逻辑 检查数组越界 如果一切无误,请尝试换一种算法 Runtime Error 运行库错误释义 运行库错误 解决办法 查看是否使用了未知代码 检查库文件引入 检查程序基本结构 检查代码运行软件(如果是比赛则不可能是这个原因) Compile Error 编译错误释义 你的程序输错啦!(指程序的语法) 解决办法 检查所有代码 检查括号匹配 检查关键字
